ABC Font Editor is a PyQt5-based GUI tool for editing .abc font files used in the games FlatOut, FlatOut 2, FlatOut: Ultimate Carnage, and FlatOut: Head On.
⚠ Work in progress — feature set not final, but fully functional.

It allows you to:

    Load .abc files alongside their texture files (.dds or .png)

    View and inspect individual glyphs with adjustable zoom

    Edit glyph coordinates, padding, widths, and other parameters

    Export glyph data to JSON (in UV or pixel coordinates) for easy external editing

    Import modified JSON back into the .abc file

    Save updated .abc files with visual verification

Designed for modders and developers who want a visual and precise way to modify bitmap font data in the FlatOut series.
